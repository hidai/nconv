## 開発メンバー向けガイド

### 新規参加者が最初に行うべきこと
1. README.mdを読む
2. DESIGN.mdで設計を理解する
3. 現在のTODOリストを確認
4. 開発環境をセットアップ (`npm install`, `npm run dev`)
5. 高優先度タスクから着手

### タスクの進め方
1. TODO項目を「進行中」に移動
2. 必要に応じて詳細なサブタスクを追加
3. 実装完了後、「完了済み」に移動
4. 技術的負債があれば TECHNICAL_DEBT.md に記録
